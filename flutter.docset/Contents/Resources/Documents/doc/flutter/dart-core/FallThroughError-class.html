FallThroughError

FallThroughError class Null safety

Error previously thrown when control reaches the end of a switch case.

The pre-2.0 Dart specification required this error to be thrown when control reached the end of a switch case (except the last case of a switch) without meeting a break or other control flow operators. That kind of fall-through was made a compile-time error Dart 2.0, so this error is no longer thrown.
