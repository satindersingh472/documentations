InitialRouteListFactory? onGenerateInitialRoutes
final

The routes generator callback used for generating initial routes if initialRoute is provided.

If this property is not set, the underlying Navigator.onGenerateInitialRoutes will default to Navigator.defaultGenerateInitialRoutes.
