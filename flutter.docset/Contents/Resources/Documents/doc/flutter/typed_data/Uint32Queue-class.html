Uint32Queue

Uint32Queue class Null safety

A QueueList that efficiently stores 32-bit unsigned integers.

For long queues, this implementation can be considerably more space- and time-efficient than a default QueueList implementation.

Integers stored in this are truncated to their low 32 bits, interpreted as an unsigned 32-bit integer with values in the range 0 to 4294967295.
