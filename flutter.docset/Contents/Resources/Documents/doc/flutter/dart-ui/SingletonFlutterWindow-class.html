SingletonFlutterWindow

SingletonFlutterWindow class Null safety

A FlutterWindow that includes access to setting callbacks and retrieving properties that reside on the PlatformDispatcher.

It is the type of the global window singleton used by applications that only have a single main window.

In addition to the properties of FlutterView, this class provides access to platform-specific properties. To modify or retrieve these properties, applications designed for more than one main window should prefer using WidgetsBinding.instance.platformDispatcher instead.

Prefer access through WidgetsBinding.instance.window or WidgetsBinding.instance.platformDispatcher over a static reference to window, or PlatformDispatcher.instance. See the documentation for PlatformDispatcher.instance for more details about this recommendation.
