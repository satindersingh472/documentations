PriorityQueue

PriorityQueue<E> class Null safety

A priority queue is a priority based work-list of elements.

The queue allows adding elements, and removing them again in priority order. The same object can be added to the queue more than once. There is no specified ordering for objects with the same priority (where the comparison function returns zero).

Operations which care about object equality, contains and remove, use Object.== for testing equality. In most situations this will be the same as identity (identical), but there are types, like String, where users can reasonably expect distinct objects to represent the same value. If elements override Object.==, the comparison function must always give equal objects the same priority, otherwise contains or remove might not work correctly.
