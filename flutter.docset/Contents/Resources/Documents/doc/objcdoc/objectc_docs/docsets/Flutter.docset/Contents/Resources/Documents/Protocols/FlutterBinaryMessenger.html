FlutterBinaryMessenger

Objective-C

@protocol FlutterBinaryMessenger <NSObject>

Swift

protocol FlutterBinaryMessenger : NSObjectProtocol

A facility for communicating with the Flutter side using asynchronous message passing with binary messages.

Implementated by:

  • FlutterBasicMessageChannel, which supports communication using structured messages.
  • FlutterMethodChannel, which supports communication using asynchronous method calls.
  • FlutterEventChannel, which supports commuication using event streams.
